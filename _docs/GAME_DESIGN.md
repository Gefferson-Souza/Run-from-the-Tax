# Resumo do Jogo
Um endless runner vertical onde o jogador foge de taxas e impostos.
- **Meta:** Acumular dinheiro para comprar upgrades (Carros, Faculdades).
- **Temas:** Brasil (Inimigos: Políticos BR) e EUA (Inimigos: Trump/ICE).
- **Mecânica:** 3 Pistas (Esquerda, Meio, Direita). Swipe para mover.
