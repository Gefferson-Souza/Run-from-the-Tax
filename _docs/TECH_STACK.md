# Tech Stack & Rules
- **Framework:** Expo (SDK 50+) Managed Workflow.
- **Language:** TypeScript (Strict Mode).
- **3D Engine:** @react-three/fiber + @react-three/drei + @react-three/rapier (Physics).
- **Styling:** NativeWind (Tailwind CSS for React Native).
- **State Management:** Zustand (Stores separadas para GameState, PlayerState, Settings).
- **Navigation:** Expo Router (File-based navigation).
- **Persistence:** MMKV (para salvar dinheiro e high score).
